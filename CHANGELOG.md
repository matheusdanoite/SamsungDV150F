# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-23

### Added
- AutoShare support for automatic photo reception.
- MobileLink support for on-demand photo browsing and downloading.
- Native iOS application structure.
- Implementation of Samsung camera network protocols.
- Basic UI with gallery and connection management.

### Changed
- Reorganized project structure for GitHub deployment.
- Moved core source files to root directory.
- Updated documentation (README improvements and translation).
