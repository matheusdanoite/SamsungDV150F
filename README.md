# Samsung Camera App

[Português Brasileiro](README.pt-br.md)

This project is a native iOS application designed to integrate with Samsung DV150F cameras.

## Features
- **AutoShare Support**: Automatically receive photos from the camera as they are taken.
- **MobileLink Support**: Browse and download photos from the camera on demand.

## Project Structure
- `CameraSamsung/`: Core source files for the application.
- `CameraSamsung.xcodeproj`: Xcode project file.

## Requirements
- iOS 15.0+
- Xcode 14.0+
- Samsung Camera with Wi-Fi support (e.g., DV150F).

## Setup
1. Open `CameraSamsung.xcodeproj` in Xcode.
2. Select the target (iOS).
3. Build and run.
