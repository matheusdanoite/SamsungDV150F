# Samsung Camera App

[English](README.md)

Este projeto é um aplicativo iOS nativo projetado para se integrar com câmeras Samsung DV150F.

## Funcionalidades
- **Suporte AutoShare**: Receba fotos da câmera automaticamente assim que forem tiradas.
- **Suporte MobileLink**: Navegue e baixe fotos da câmera sob demanda.

## Estrutura do Projeto
- `CameraSamsung/`: Arquivos fonte principais do aplicativo.
- `CameraSamsung.xcodeproj`: Arquivo de projeto do Xcode.

## Requisitos
- iOS 15.0+
- Xcode 14.0+
- Câmera Samsung com suporte Wi-Fi (ex: DV150F).

## Instalação
1. Abra `CameraSamsung.xcodeproj` no Xcode.
2. Selecionar o alvo (iOS).
3. Compilar e rodar (Build and run).
